# SALES-PERFORMANCE-DASHBOARD
📊 Sales Dashboard (Power BI)

🔍 Overview

This project is an interactive Sales Dashboard built using Power BI to analyze business performance across different countries, products, and customers.

It helps in understanding key sales metrics, identifying top-performing products, and tracking trends over time.

---

🎯 Key Features

- 📌 Total Sales, Orders, Customers & Products KPIs
- 🌍 Country-wise Sales Analysis
- 📈 Year-wise Sales Trend
- 🛒 Product Category Distribution
- 🏆 Top 5 Customers Analysis
- 📊 Top-Selling Products
- 🎛️ Interactive Filters (Country, City, Date)

---

🛠️ Tools & Technologies Used

- Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Excel / CSV Dataset
